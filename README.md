# Amazon-Inspired Frontend Website

A responsive frontend practice project inspired by the Amazon homepage UI, built using pure HTML and CSS.

## 🚀 Live Demo
[View Website](https://effortless-melba-bfe4d9.netlify.app)

## 📌 Features
- Responsive navigation bar
- Search bar UI
- Product sections
- Hero/banner section
- Footer section
- Clean layout using Flexbox
- Mobile-friendly design

## 🛠️ Technologies Used
- HTML5
- CSS3

## 🎯 Purpose of Project
This project was created to improve frontend development skills and practice:
- Layout design
- Responsive styling
- Flexbox
- UI structuring
- Website deployment using Netlify
